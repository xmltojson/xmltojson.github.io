Tired of writing test automation code just to check if a button works? 🧪

We're thrilled to present **Application Verifier** — a modern, browser-based BDD scenario testing tool that lets you write and execute web app tests in plain English (or Ukrainian!) — no coding, no installation, no server required.

🎁 WHAT'S INSIDE:

🎯 **BDD-Style Scenarios** — Write tests using Given/When/Then/And syntax
🇺🇦 **Bilingual Steps** — Native Ukrainian & English keywords in every pattern
📋 **Scenario Groups** — Organize tests into collapsible, reorderable groups
▶️ **Flexible Execution** — Run all, selected, single group, or one step
🔀 **Enable/Disable Toggles** — Skip steps or groups without deleting them
📥 **Multi-Format Import** — JSON, YAML, and Gherkin `.feature` files
📤 **Multi-Format Export** — Share scenarios across teams and tools
📎 **File Upload Testing** — Built-in file manager for upload-form tests
🖼️ **Live Preview** — Watch tests run in real-time inside embedded iframe
📊 **Real-Time Reports** — Stats, progress bars & detailed test reports
📝 **Execution Journal** — Color-coded timestamped log of every action

🖥️ TECHNICAL HIGHLIGHTS:

✓ Pure HTML/CSS/JavaScript — Zero backend, single self-contained file
✓ 25+ natural-language step patterns (English + Ukrainian simultaneously)
✓ Native `checkVisibility` API for robust element detection
✓ React/Vue-compatible input setters via `Object.getOwnPropertyDescriptor`
✓ `DataTransfer` API for realistic file upload simulation
✓ Configurable per-step timeouts (1–300s) with abort control
✓ localStorage persistence for scenarios, files & settings
✓ Fully responsive design (mobile, tablet, desktop)
✓ Custom dialog system (no ugly browser alerts!)
✓ Split-view layout with horizontal/vertical orientation toggle
✓ 100% privacy — everything runs locally, nothing uploaded anywhere
