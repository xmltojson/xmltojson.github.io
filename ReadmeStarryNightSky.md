### Visual Elements
- **Twinkling Stars** - Hundreds of stars with varying sizes and realistic twinkling animations
- **Shooting Stars** - Random meteors streaking across the night sky
- **Constellations** - 6 famous constellations with interactive information
  - Ursa Major (Big Dipper)
  - Orion (The Hunter)
  - Cassiopeia (The Queen)
  - Leo (The Lion)
  - Cygnus (The Swan)
  - Scorpius (The Scorpion)
- **Moon** - Detailed moon with crater details
- **Aurora Borealis** - Animated northern lights effect
- **Nebulae** - Subtle colorful nebula clouds in the background

### Interactive Controls
- Toggle stars visibility
- Toggle shooting stars
- Toggle constellations display
- Toggle moon
- Toggle aurora effect
- Reset all settings to default

### State Persistence
- All user preferences saved to localStorage
- Shooting star counter persists across sessions
- Settings survive page refresh

### Responsive Design
- Fully responsive layout for all screen sizes
- Optimized for both portrait and landscape orientations
- Touch-friendly controls for mobile devices
- Adaptive UI elements
