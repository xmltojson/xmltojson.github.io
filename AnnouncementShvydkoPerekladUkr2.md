Дорога спільното,

Ми раді повідомити про велике оновлення **ШвидкоПереклад** — безкоштовного AI-перекладача, створеного спеціально для українців, які облаштовують життя за кордоном! 🇺🇦

Прислухавшись до ваших відгуків, ми наповнили цей реліз трьома потужними новими функціями, які роблять переклад швидшим, доступнішим і кориснішим у повсякденних ситуаціях.

🎁 **ЩО НОВОГО У v3.0:**

🇨🇿 **Підтримка чеської мови** — Повний переклад, OCR та голос для Čeština
⭐ **Вкладка збережених перекладів** — Збереження в один клік з перемикачем (✓ Збережено)
🔊 **Розумне голосове відтворення** — Прослухайте будь-який переклад, натисніть ще раз, щоб зупинити
⏹ **Зупинка будь-коли** — Кожна кнопка озвучення тепер є перемикачем з візуальним відгуком
🎯 **Автовизначення голосу** — Обирає найкращий доступний голос для кожної мови
📋 **Дії для кожного елемента** — Відкрити, Копіювати, Озвучити, Видалити зі збережених та історії
🎨 **Пульсуюча червона анімація** — Бачте точно, який текст озвучується
ℹ️ **Діагностика голосу** — Інформаційне вікно показує, які TTS-голоси підтримує ваш пристрій

🖥️ **ТЕХНІЧНІ ОСОБЛИВОСТІ:**

✓ Чистий HTML/CSS/JavaScript — Жодних залежностей (окрім Tesseract.js для OCR)
✓ Надійна реалізація Web Speech API з виправленням 15-секундного обмеження Chrome
✓ Асинхронне завантаження голосів із розумним підбором резервних варіантів
✓ IndexedDB для зберігання зображень, LocalStorage для налаштувань
✓ Автоматична зупинка при перемиканні вкладок, приховуванні сторінки та подіях очищення
✓ Власна модальна система замість браузерних сповіщень
✓ Повністю адаптивний — мобільні, планшети, десктопи та ландшафтний режим
✓ 100% на стороні клієнта — ваші дані ніколи не покидають ваш браузер
