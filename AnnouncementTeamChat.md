Dear Community,
Tired of clunky chat tools that need accounts, servers, and IT setup? 💬

We're excited to introduce **Team Chat** — a beautiful, zero-setup team messenger that runs entirely in your browser. No signups, no installs, no backend. Just share a link and start chatting.

🎁 WHAT'S INSIDE:

💬 **Channels & DMs** — Topic-based conversations and private 1-on-1 chats
✏️ **Edit & Delete** — Fix typos or remove messages with full transparency
📎 **File Sharing** — Drag, drop, or paste images and documents
🖼️ **Image Lightbox** — Beautiful full-screen image viewer
🟢 **Online Indicators** — See who's active in real-time
🔄 **Cross-Browser Sync** — Messages sync across all devices automatically
🔗 **Instant Invites** — One-click shareable links for teammates
🌙 **Dark Theme UI** — Clean, modern interface inspired by top messengers

🖥️ TECHNICAL HIGHLIGHTS:

✓ Pure HTML/CSS/JavaScript — Single file, no dependencies
✓ Serverless architecture via JSONBlob cloud storage
✓ Smart merge algorithm for concurrent edits
✓ Offline-first with localStorage persistence
✓ Fully responsive — mobile, tablet, desktop
✓ Touch-friendly with mobile gestures
✓ Keyboard shortcuts for power users

🎯 PERFECT FOR:

• Small teams who need quick async chat
• Family or friend groups staying connected
• Project collaboration without IT overhead
• Classroom and study group communication
• Hackathon teams needing instant setup
• Anyone wanting full data ownership

🔗 Try it live: https://teamchatjs.github.io
