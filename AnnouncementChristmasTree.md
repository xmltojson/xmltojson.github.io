🎁 WHAT'S INSIDE:

🎄 Canvas-Rendered Tree — A detailed, layered tree drawn in real time
❄️ Falling Snow — Realistic, continuously generated snowfall
💡 Twinkling Lights — 50 spiraling, color-cycling lights that glow and flicker
🎨 Animated Ornaments — Glossy, pulsing baubles in 8 vibrant colors
🎁 Add Ornament — Decorate the tree dynamically with a burst of floating notes
⭐ Glowing Star Topper — A pulsing golden star crowns the tree
🎵 Advanced Music System — A built-in synthesizer playing 5 full carols
🎼 Song Selector — Jingle Bells, O Christmas Tree, Silent Night & more
📊 Music Visualizer — Animated equalizer bars synced to the music
🎁 Bouncing Presents — Wrapped gifts with golden bows at the tree base

🖥️ TECHNICAL HIGHLIGHTS:

✓ Pure HTML/CSS/JavaScript — No dependencies
✓ Canvas-based tree, lights & ornament rendering
✓ Web Audio API multi-track music synthesis (melody, harmony & bass)
✓ Custom instruments: bells, glockenspiel, pads, bass & sleigh bells
✓ Reverb & dynamics compression for a polished sound
✓ Fully responsive design
✓ Mobile & desktop optimized
✓ Auto-rotating carol playlist
