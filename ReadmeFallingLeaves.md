# Falling Leaves

A beautiful, interactive autumn scene with realistic falling leaves, dynamic wind effects, and full responsive support.

## Demo

[Live Demo](https://fallingleavesj.github.io)

## Features

### Visual Effects
- Realistic falling leaves with multiple leaf types
- Dynamic wind system with gusts and direction changes
- Animated sun with glowing effect
- Floating clouds drifting across the sky
- Autumn trees with colorful crowns
- Ground with leaf piles
- Sparkle effects on click/touch

### Customizable Settings
- **Leaf Count** - Adjust number of leaves (10-200)
- **Wind Speed** - Control wind intensity
- **Fall Speed** - Set how fast leaves fall
- **Spin Speed** - Adjust leaf rotation speed
- **Leaf Size** - Change leaf dimensions
- **Show/Hide Trees** - Toggle tree visibility
- **Show/Hide Clouds** - Toggle cloud visibility
- **Wind Gusts** - Enable/disable random wind gusts

### State Persistence
- All settings automatically saved to localStorage
- Settings restored on page reload
- Reset button to restore defaults

### Responsive Design
- Full support for mobile devices
- Portrait and landscape orientations
- Collapsible control panel
- Touch interactions (tap to create leaves)
- Optimized for small screens

### Interactive Elements
- Click/tap anywhere to spawn leaves with sparkle effects
- "Leaf Burst" button for dramatic leaf explosion
- Wind indicator showing current wind direction and strength

## Installation

1. Clone the repository:
```bash
git clone https://github.com/fallingleavesj/fallingleavesj.github.io.git
```

2. Open `index.html` in your browser

## Usage

- **Adjust Settings**: Use the control panel sliders and checkboxes
- **Create Leaves**: Click or tap anywhere on the scene
- **Leaf Burst**: Click the "Leaf Burst" button for a dramatic effect
- **Toggle Panel**: Click the header to collapse/expand controls
- **Reset**: Click "Reset Settings" to restore defaults

## Technologies

- HTML5
- CSS3 (Animations, Gradients, Flexbox)
- JavaScript (ES6+, Canvas API)
- LocalStorage API
