# 🌻 ШвидкоПереклад

> Free AI translator with specialized context support for official documents, medical terms, and legal paperwork.

**🔗 [Open Application](https://shvydkopereklad.github.io/)**

## ✨ Features

- 🌐 **Free translation** — no API keys, no registration, no limits
- 🎯 **Context-aware** — specialized modes for legal, medical, official documents, education, and banking
- 📷 **Photo OCR** — recognize and translate text from document photos directly in your browser
- 🎤 **Voice input** — speak instead of typing
- 🔊 **Text-to-speech** — listen to translations
- 💬 **Phrasebook** — useful phrases for emergencies, medicine, documents, housing, and work
- 📚 **History** — automatic saving of all translations
- ⭐ **Favorites** — bookmark important translations
- 🔒 **Privacy-first** — all data stored locally in your browser
- 📱 **Mobile-optimized** — works beautifully on phones, tablets, and desktops
- 🌍 **4 languages** — Ukrainian 🇺🇦 · English 🇬🇧 · Polish 🇵🇱 · German 🇩🇪

## 🚀 How It Works

### Translation Providers (Automatic Fallback)
1. **Google Translate** — primary provider (fastest, best quality)
2. **MyMemory API** — fallback (5000 words/day free)
3. **LibreTranslate** — secondary fallback (public instances)

If one provider fails, the next one automatically takes over.

### OCR (Optical Character Recognition)
Powered by [Tesseract.js](https://tesseract.projectnaptha.com/) — runs entirely in your browser. Images never leave your device.

## 📖 Usage

1. Open the [application](https://shvydkopereklad.github.io/)
2. Select source and target languages
3. Choose a context (optional, improves accuracy)
4. Type, paste, or speak your text
5. Click **Перекласти** (Translate)

### Photo Translation
1. Go to the **📷 Фото** tab
2. Upload a photo or take one with your camera
3. Select source and target languages
4. Click **🔍 Розпізнати і перекласти**

## 🛠️ Tech Stack

- Pure HTML, CSS, and JavaScript (no build step, no dependencies)
- [Tesseract.js](https://github.com/naptha/tesseract.js) for OCR
- Web Speech API for voice input/output
- LocalStorage for data persistence
- Progressive Web App ready
