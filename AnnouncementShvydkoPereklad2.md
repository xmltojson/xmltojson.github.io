Dear Community,

We're excited to announce a major update to **ШвидкоПереклад** — the free AI translator built especially for Ukrainians navigating life abroad! 🇺🇦

After listening to your feedback, we've packed this release with three powerful new features that make translation faster, more accessible, and more useful for everyday situations.

🎁 **WHAT'S NEW IN v3.0:**

🇨🇿 **Czech Language Support** — Full translation, OCR & voice for Čeština
⭐ **Saved Translations Tab** — One-click save with toggle (✓ Збережено)
🔊 **Smart Voice Playback** — Listen to any translation, click again to stop
⏹ **Stop Anytime** — Every speak button is now a toggle with visual feedback
🎯 **Voice Auto-Detection** — Picks the best available voice for each language
📋 **Per-Item Actions** — Open, Copy, Speak, Delete from saved & history
🎨 **Pulsing Red Animation** — See exactly which text is being spoken
ℹ️ **Voice Diagnostics** — Info dialog shows which TTS voices your device supports

🖥️ **TECHNICAL HIGHLIGHTS:**

✓ Pure HTML/CSS/JavaScript — Zero dependencies (except Tesseract.js for OCR)
✓ Robust Web Speech API implementation with Chrome 15s-cutoff fix
✓ Async voice loading with smart fallback matching
✓ IndexedDB for image storage, LocalStorage for settings
✓ Auto-stop on tab switch, page hide & cleanup events
✓ Custom modal system replacing browser alerts
✓ Fully responsive — mobile, tablet, desktop & landscape
✓ 100% client-side — your data never leaves your browser
