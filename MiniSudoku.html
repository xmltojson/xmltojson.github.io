How to Play Mini Sudoku
Basic Rules
  • Fill the 6×6 grid with numbers 1-6
  • Each row must contain all numbers 1-6
  • Each column must contain all numbers 1-6
  • Each 2×3 block must contain all numbers 1-6
  • No number can repeat in any row, column, or block

How to Play
  • Click empty cell to select
  • Click number (1-6) to place
  • Gray cells are fixed
  • Use Notes for reminders
  • Hint reveals one answer
  • Check verifies solution
