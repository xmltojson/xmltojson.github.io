# Drawing Canvas

A powerful, browser-based drawing and painting application with a modern interface. Create digital artwork directly in your browser with no installation required.

## ‚ú® Features

### Drawing Tools
- **Pencil** - Precise line drawing
- **Brush** - Smooth strokes with adjustable hardness
- **Marker** - Semi-transparent overlay strokes
- **Spray** - Airbrush-style painting
- **Eraser** - Remove parts of your artwork

### Shape Tools
- Line
- Rectangle
- Circle/Ellipse
- Triangle
- Fill shape option with customizable stroke width

### Additional Tools
- **Fill Bucket** - Flood fill areas with color
- **Color Picker** - Sample colors from canvas
- **Text Tool** - Add text with customizable font, size, and style

### Brush Settings
- Adjustable size (1-100px)
- Opacity control (1-100%)
- Hardness/softness control
- Real-time brush preview

### Color Options
- Primary and secondary color selection
- Full color picker
- Pre-defined color palette (48 colors)

### Canvas Management
- Multiple preset canvas sizes (Standard, Full HD, Square, Portrait)
- Custom canvas dimensions
- Background color selection
- Zoom in/out with reset option
- Clear canvas function

### File Operations
- **Save** - Store artwork in browser's local storage
- **Load** - Open previously saved artwork
- **Download** - Export as PNG, JPEG, or WebP
- **Upload** - Import images to edit or use as canvas
- **Auto-save** - Automatic recovery of work in progress

### History
- Unlimited undo/redo (up to 50 states)
- Full history tracking

### User Interface
- Dark theme modern interface
- Responsive design for desktop and mobile
- Touch screen support
- Collapsible sidebar for mobile devices
- Toast notifications for user feedback

## üöÄ Getting Started

### Online Usage
Simply visit [https://drawingcanvas.github.io](https://drawingcanvas.github.io) to start drawing immediately.

### Local Installation

1. Clone the repository:
```bash
git clone https://github.com/drawingcanvas/drawingcanvas.github.io.git
```

2. Navigate to the project directory:
```bash
cd drawingcanvas.github.io
```

3. Open `index.html` in your web browser.
