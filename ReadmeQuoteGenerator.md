## ‚ú® Features

### Core Functionality
- **50 Inspirational Quotes** - Curated collection from famous authors, leaders, and thinkers
- **Random Generation** - No repeats until all quotes are shown
- **Category Tags** - Each quote categorized (Success, Wisdom, Hope, Courage, etc.)

### User Experience
- **Favorites System** - Save your favorite quotes with local storage persistence
- **Text-to-Speech** - Listen to quotes read aloud
- **Copy to Clipboard** - One-click quote copying
- **Social Sharing** - Share directly to Twitter, Facebook, WhatsApp, and LinkedIn

### Visual Design
- **6 Color Themes** - Purple, Pink, Blue, Green, Sunset, and Pastel
- **Animated Background** - Floating shapes with smooth animations
- **Stylish Typography** - Google Fonts (Playfair Display, Poppins, Dancing Script)
- **Glass-morphism Card** - Modern frosted glass effect
- **Smooth Transitions** - Fade animations for quote changes

### Responsive Design
- **Portrait Mode** - Vertical stacked layout
- **Landscape Mode** - Horizontal optimized layout
- **Mobile Friendly** - Works on all screen sizes
- **Dark Mode Support** - Follows system preference
- **Print Styles** - Clean output for printing
