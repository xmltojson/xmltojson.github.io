# Analog Clock

A beautiful animated analog clock with customizable themes, responsive design, and persistent settings.

![Analog Clock](https://analogclockj.github.io/screenshot.png)

## Demo

[Live Demo](https://analogclockj.github.io)

## Features

### Themes
- **Midnight** - Deep blue with red accents
- **Ocean** - Calming cyan and teal
- **Forest** - Natural greens
- **Sunset** - Warm oranges and pinks
- **Lavender** - Soft purples
- **Minimal** - Clean black and white
- **Gold** - Elegant gold tones

### Customization Options
- Toggle number visibility
- Switch between Arabic and Roman numerals
- Smooth or ticking second hand
- Animated glow effect
- 12/24 hour digital time format

### Responsive Design
- Adapts to small screens
- Supports portrait and landscape orientations
- Optimized for mobile, tablet, and desktop

### Persistent Settings
All settings are automatically saved to localStorage and restored on page reload.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/analogclockj/analogclockj.github.io.git
```

2. Open `index.html` in your browser.

## Usage

- Click on colored circles to change themes
- Use toggle buttons to customize:
  - **Numbers** - Show/hide clock numbers
  - **Roman** - Switch to Roman numerals
  - **Smooth** - Enable smooth second hand movement
  - **Glow** - Toggle glow animation effect
  - **24H** - Switch between 12/24 hour format
