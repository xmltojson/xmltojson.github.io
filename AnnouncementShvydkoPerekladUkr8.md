🗣️ **Жива розмова** — Абсолютно нова вкладка «Розмова», яка слухає, перекладає та автоматично озвучує результат
🎙️ **Інтерфейс «Натисни й говори»** — Велика зручна кнопка мікрофона з наочними станами: очікування, прослуховування та обробка
📝 **Живе розпізнавання** — Спостерігайте, як ваші слова з'являються на екрані під час мовлення, з проміжними результатами в реальному часі
🔁 **Безперервний режим** — Після озвучення перекладу застосунок автоматично продовжує слухати наступну фразу — ідеально для діалогу
🔊 **Перемикач автоозвучення** — Вмикайте або вимикайте автоматичне озвучення одним дотиком
⇄ **Миттєвий обмін мовами** — Міняйте мову-джерело та мову перекладу місцями просто під час розмови
💬 **Журнал розмови** — Кожен обмін репліками зберігається з можливістю повторно озвучити чи скопіювати
🔂 **Кнопка повтору** — Прослухайте останній переклад ще раз будь-коли

🖥️ **ТЕХНІЧНІ ОСОБЛИВОСТІ:**

✓ Чистий HTML/CSS/JavaScript — без збірки, без фреймворків
✓ Повний цикл: розпізнавання мовлення (Web Speech) → переклад через кілька провайдерів → синтез мовлення
✓ Використовує надійний ланцюжок резервних сервісів (Google → MyMemory → LibreTranslate)
✓ Покращений рушій озвучення з колбеком `onDone`, щоб прослуховування відновлювалося плавно після завершення мовлення
✓ Безперервний режим розумно призупиняється, коли ви залишаєте вкладку, щоб економити ресурси
✓ Інтегровано з історією перекладів та враховує денні ліміти використання
✓ Налаштування живої розмови зберігаються через LocalStorage
✓ Коректна обробка помилок доступу до мікрофона та непідтримуваних браузерів
✓ Повністю адаптивний — зона мікрофона гарно масштабується на мобільних екранах
✓ Україномовний інтерфейс, що залишається теплим, зрозумілим і дружнім
