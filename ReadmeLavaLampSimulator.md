# Lava Lamp Simulator

A mesmerizing, interactive lava lamp simulation with realistic blob physics, metaball effects, and smooth animations.

## Demo

**Live Demo:** [https://lavalampsimulator.github.io](https://lavalampsimulator.github.io)

## Features

### Visual Effects
- **Metaball rendering** - Blobs seamlessly merge and separate
- **Gradient colors** - Temperature-based color interpolation
- **Glass reflections** - Realistic lamp glass with highlights and shine
- **Bubble particles** - Small bubbles rise from heated blobs
- **Heat glow** - Animated glow effect at the lamp base
- **High DPI support** - Crisp rendering on retina displays

### Physics Simulation
- **Temperature dynamics** - Blobs heat at the bottom, cool at the top
- **Buoyancy system** - Hot blobs rise, cool blobs sink
- **Blob collisions** - Realistic collision detection with momentum exchange
- **Organic movement** - Noise-based wandering for natural motion
- **Squish deformation** - Blobs deform when hitting boundaries
- **Viscosity simulation** - Fluid-like damping effects

### Controls
| Control | Description |
|---------|-------------|
| Blobs | Number of lava blobs (4-12) |
| Speed | Animation speed |
| Heat | Heating intensity affecting rise/fall rate |
| Size | Blob size multiplier |
| Gooey | Metaball merge threshold |
| Colors | Two-color gradient picker |
| Liquid | Background liquid color |
